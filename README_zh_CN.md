# 思源白板插件(预览版)

> 基于tldraw的白板工具。

Tldraw Version: 2.0.0-beta.1

## 安装指南
1. 从集市安装后开启此插件，重启或刷新思源后，可在左下角找到白板的Dock。
2. 在白板Dock中点击+，输入名称，创建白板
3. 点击此名称的白板，即可显示内容

## 使用方法

1. Tldraw的基础使用方法可在网上搜索其他使用方法
2. 点击块菜单中的“复制嵌入块”或“复制引用块”，在白板中粘贴可生成嵌入块或引用块。
